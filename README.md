# Stagentic Claude Code Marketplace

The Claude Code plugin marketplace for the [stagentic](https://github.com/stagentic) family of tools.

## Add the marketplace

In Claude Code:

```
/plugin marketplace add stagentic/stagentic-cc-marketplace
```

## Install a plugin

```
/plugin install stagentic-promptbook@stagentic
```

## Available plugins

- [`stagentic-promptbook`](https://github.com/stagentic/stagentic-promptbook) — PlantUML activity diagrams as Claude Code skill workflows.

More plugins from the stagentic family will be listed here as they ship.
